Система добавления данных о качественных показателях железорудного концентрата.

список библиотек с версиями в requirements.txt

REST API:
  Пользователю предлагается ввести данные о 
  "Наименование сырья"
  "Содержание железа"
  "Содержание алюминия"
  "Содержание кремния"
  "Содержание кальция"
  "Содержание серы"
  "Дата добавления"
